## 개요

**천부태을(天符太乙)**은 한민족 최고(最古)의 경전인 천부경 81자의 수리 체계와, 동양 최고(最高) 수학 점술인 태을신수(太乙神數)를 하나의 엔진으로 결합하여, 개인의 사주·체질·운세·건강·비즈니스 전략을 통합 분석하는 Python 프로그램입니다.

### 핵심 공식
```text
天符經 81字 (體) ─── 선천·후천 본명수, 구궁 배속, 사상체질│ 
├──── 交叉分析 (體用合一) ────┐  │
│ 太乙神數 (用) ─── 세국(世局)×명국(命局) 이중 레이어 ── 개인화 승산지수 │ 
├──── 太乙六壬 (占) ──── 4중 포국(歲·月·日·時)  │ 
└──── 健康分析 (健) ──── 삼혼·삼신·오장·사상체질·인산의학
```

## 핵심 원리

천부경(體)은 **"당신은 누구인가"**를 알려주고, 태을신수(用)는 **"지금 우주가 어떻게 움직이고 있는가"**를 알려줍니다. 이 둘을 합치면 **"당신이 지금 이 우주 속에서 어떤 위치에 있는가"**가 나옵니다. 앞으로 버전 개선에도 이 핵심 원리를 주지를 유지할 계획입니다.



### 천부경(天符經)

천부경은 단군(檀君)이 전한 81자의 경문으로, 한민족 사상의 원류로 주장되고 칭해집니다. 우주의 생성·변화·귀환의 원리를 一에서 十까지의 수(數)와 天地人 삼재(三才)로 '압축한' 경전입니다.

인산(仁山) 김일훈(金一勳) 선생은 저서 『신약본초(神藥本草)』 「신서 천부경 주해(神書天符經註解)」에서 다음과 같이 밝혔습니다:

> **"八十一字天符經은 宇宙의 神數이니 太乙神數經은 天符經에 대한 解釋이 完全하니라."**
>
> *(팔십일자 천부경은 우주의 신수이니 태을신수경은 천부경에 대한 해석이 완전하니라.)*

> **"太字는 八十이오 乙字는 一也니라."**
>
> *(태(太)자는 팔십(八十)이오 을(乙)자는 일(一)이니라.)*

즉, **太乙 = 八十一 = 天符經 81字**입니다. 태을신수는 곧 천부경 81자의 수리적 전개이며, 이 프로그램은 인산 선생의 이 핵심 통찰을 코드로 구현한 것입니다.



그리고 인산 선생의 천부경 주해 가이드에 따라 다음의 원칙도 적용 되었습니다 :  

**첫째**, 천부경 주해의 "구궁육진법(九宮六進法)"과 "구궁삼익진법(九宮三翼陣法)"의 1천록·2안손·3식신·4증파·5귀귀·6합식·7친귀·8관인·9퇴식 9궁 길흉 체계가 태을신수의 태을행궁 9궁(건1·이2·간3·진4·중5·태6·곤7·감8·손9)과 정확히 일대일로 대응하며, 천부경의 "용변부동본"이 태을의 "불입중궁(不入中宮)" 원칙으로 실현됩니다.

**둘째**, 천부경 주해의 "순행역행하니 일년공전이오 순환무단하니 일일자전"이 태을신수의 양둔(1궁기→순행)·음둔(9궁기→역행) 및 년월일은 양둔·시는 음양둔으로 나누는 이중 운행 체계의 직접적 근거이며, 천부경의 "천도양선어음 지도음선어양"이 양둔은 건곤중류·음둔은 간손중류로 구현됩니다.

**셋째**, 천부경 주해의 "천일일 임자일·지일이 정사이·인일삼 갑인삼" 삼재 체계가 태을의 천목(文昌·형혹성정·화)→객목(始擊·진성정·토)→계신(세성지사·목) 삼포인터로 동적 변환되며, "석삼극무진본"의 1→3 분화가 태극→이목→사장→팔장의 1→2→4→8 이진 분화로 전개됩니다.

**넷째**, 천부경 주해의 "삼사성환" 3×4=12지와 "대삼합육" 육합 체계가 태을 16신(12지+건곤간손 4우궁)으로 확장되고, "오원육기 360년=5×72=6×60" 주기 구조가 천부경의 "일적십거(1→10)"·"대삼합육(6합)"·"생칠팔구운(8궁×9=72)"의 수리를 시간축 위에서 중첩적으로 운용합니다.

**다섯째**, 천부경 주해에서 강태공이 81진 중 길36진(1·3·6·8궁)과 흉45진(2·4·5·7·9궁)으로 승전한 구궁삼익진법의 실전 사례가 태을신수의 관·수·격·핍·엄·격·제협·사곽고 격국 판정 체계와 팔문(개·휴·생 대길, 상·두·사 대흉) 길흉 판단으로 세분화되어, 천부경의 정적 길흉 구조가 태을의 적년 연산을 통해 임의 시점에 대입 가능한 동적 예측 엔진으로 완성됩니다.

### 태을신수(太乙神數)

태을신수는 중국 고대 **三式(삼식: 태을·기문·육임)** 중 최상위 술수로, 천도(天道)의 운행을 수리로 포국(布局)하여 국가·시대·개인의 운수를 판단하는 체계입니다.

핵심 연산:
- **태을적년수**: 입성수(10,153,917) + 서기 연도
- **태을재궁**: 적년수를 360→72→24로 나누어 태을의 궁 위치 산출
- **16신 배치**: 태을·천목(文昌)·객목(始擊)·계신(計神) 등 16신의 궁 배정
- **주객산(主客算)**: 주산·객산·정산으로 승패와 승산지수 계산
- **격국(格局)**: 태을과 16신의 상호 위치로 길흉 판정 (和·掩·提挾·格·擊·四郭固 등)
- **팔문(八門)**: 개·휴·생·상·두·경·사·경 8문을 구궁에 배치

### 인산의학과 사상체질

인산 김일훈 선생의 의학 체계에서는 혈액형을 사상체질과 연결합니다:
- **A형 → 太陰人(木)**: 간대폐소(肝大肺小)
- **B형 → 少陰人(水)**: 신대심소(腎大心小)
- **O형 → 少陽人(火)**: 심대신소(心大腎小)
- **AB형 → 太陽人(金)**: 폐대간소(肺大肝小)

본 프로그램은 이 체질 분류를 오행·년간·시운과 교차하여 개인별 건강 취약점과 양생법을 산출합니다.

### 고영창(高永昌) 만세력

본 프로그램의 사주(四柱) 산출은 **고영창 님의 「진짜 만세력(CAL20000)」** 알고리즘을 기반으로 합니다. 이 만세력은 다음과 같은 이유로 가장 정확합니다:

- 그레고리력 전체 범위(기원전~서기 9999년)를 수학적으로 완전하게 커버
- 절기(節氣) 시각을 천문학적 태양 황경(黃經) 기준으로 정밀 계산
- 대한민국 표준시(KST, UTC+9) 기준 절기 교차점을 초(秒) 단위까지 산출
- 기존 만세력의 "표 기반 참조" 방식이 아닌, 순수 수식 기반 연산으로 오차 최소화
- 월건(月建) 판정 시 절기 정확 시각을 사용하여 월주 오류 제거

## 특징

### 세계 최초 — 體用合一 이중 레이어

기존의 사주·태을 프로그램은 각각 독립적으로 작동합니다. 본 프로그램은 **세국(世局, 당해 연도 포국) × 명국(命局, 생년 포국)**을 교차하여 개인별 차등화된 결과를 산출하는 이중 레이어 구조를 구현한 최초의 프로그램입니다.

### 4대 분석 모드

| 모드 | 내용 |
|------|------|
| **1) 종합 분석** | 천부경 + 태을신수 + 교차분석 + 건강 + 10년 예측 |
| **2) 태을육임 점술** | 4중 포국(歲·月·日·時) 실시간 점사 |
| **3) 건강 분석** | 삼혼·삼신·오장·사상체질·본심본태양·산삼 참조 |
| **4) 비즈니스 리포트** | 승산지수 기반 전략 방위·리스크·Executive Summary |

### 개인화 교차 분석 (v2.1 신규)

- 생년 간지·납음오행 산출
- 세궁 오행 vs 명궁 오행 상생상극 판정
- 납음오행 vs 세궁 오행 교차 보정
- 격국 이중 판정 (세국 격국 × 명국 격국 → 최종 개인 격국)
- 세국·명국 주객산 가중 합산 (60:40)으로 개인화 승산지수 산출
- 유년괘(流年卦): 생년 기반 + 나이 → 64괘 배당

### 24방위 방위도

- 외환(16신 배치), 중환(팔문 길흉 방위), 내환(주기·객기) 3중 구조
- Blue Zone(유망 방위) / Red Zone(주의 방위) 시각적 표시
- 문창(文昌)·계신(計神) 간신(間神) 위치 표기

### 인산의학 건강 분석

- 삼혼(天魂·地魂·人魂) 상생상극 판정
- 삼신(天神·地神·人神) 시운 영향 분석
- 오장(肝·心·脾·肺·腎) 왕상휴수사(旺相休囚死) 상태 + 시운 영향
- 사상체질-년간 교차 (相生·相剋·比和 판정)
- 본심본태양(本心本太陽) 심장 상태
- 산삼 참조: 년간별 산삼 생장 방위와 보음보양 장부

## 파일 구성

| 파일 | 크기(참고) | 역할 |
|------|-----------|------|
| `main.py` | ~8KB | 메인 진입점. 입력 수집, 모드 분기, `run_full_analysis()` 출력 |
| `integration.py` | ~12KB | **v2.1 핵심.** 세국×명국 이중 레이어 교차 엔진. `IntegrationEngine` 클래스 |
| `cheonbu_core.py` | ~23KB | 천부경 엔진. 사주 산출(고영창 만세력), 본명수, 구궁 배속, 혈액형 분석 |
| `taeul_core.py` | ~25KB | 태을신수 엔진. 적년수·재궁·16신·격국·주객산·팔문·승산지수 |
| `constants.py` | ~28KB | 60갑자, 천간·지지, 오행, 상생·상극, 납음, 구궁, 16궁, 64괘, 계신 테이블 |
| `health_module.py` | ~10KB | 건강 분석. 삼혼·삼신·오장·시운·사상체질 교차·산삼 참조 |
| `divination.py` | ~9KB | 태을육임 점술. 4중 포국(歲·月·日·時), 점사 카테고리별 판단 |
| `interpreter.py` | ~7KB | 해석기. 격국 해설, 비즈니스 리포트 생성, 조사 처리 |
| `renderer.py` | ~15KB | 출력기. `Renderer` 클래스 — 게이지 바, 방위도, 건강 리포트, 점수 요약 |
| `utils.py` | ~3KB | 유틸리티. 날짜·시간·혈액형·이름 입력 검증 |

## 설치

### 요구사항

- **Python 3.8 이상** (3.10+ 권장)
- 외부 라이브러리 의존성 **없음** (Pure Python, 표준 라이브러리만 사용)

### 설치 방법

```bash
# 저장소 클론
git clone https://github.com/myzein/CheonbuTaeul.git
cd CheonbuTaeul

# 바로 실행 (별도 설치 불필요)
python main.py

```



### 환경 설정

- 터미널/콘솔의 인코딩이 **UTF-8**이어야 한자(漢字)와 특수 기호가 정상 출력됩니다.
- Windows의 경우: `chcp 65001` 실행 후 사용하거나, Windows Terminal 사용을 권장합니다.

- **양력(그레고리력)** 기준입니다. 음력 변환 불필요.
- **서머타임(DST) 보정 불필요.** 생시는 태양시 기준 시진(時辰) 배속이므로 1-2시간 편차는 동일 시진 내 포함.
- 1961년 이전 출생자는 한국 서머타임 시행 시기(1948~1961)에 해당하므로, "실제 태양시 기준 생시"를 입력하는 것을 권장합니다.

## 제한사항 및 로드맵

### ⚠️ 현재 버전의 기술적 한계

본 프로그램은 **v2.1 (실질적으로 v1.2 수준)**이며, 완전한 v2.0을 향해 가는 **중간 과정**입니다. 아직 완벽한 버전이 아니며, 다음과 같은 제한사항과 개선 과제가 존재합니다:

- 천부경 체(體) 분석의 본명수·선천운·후천운 산출 알고리즘이 아직 단순화되어 있어, 천부 점수의 변별력이 낮음 (14~28점 범위에 집중)
- 태을신수 인도명법(人道命法) — 개인 명궁·신궁·대유진한·유년괘의 원전 완전 구현이 미완 (현재는 Method A: 세국×명국 교차 방식으로 개인화하고 있으며, 원전의 완전한 인도명법은 v3.0에서 구현 예정)
- 사주(四柱) 산출 시 일주(日柱) 정밀도가 만세력 원본 대비 검증 미완 — 일부 경계일(절기 교차일)에서 오차 가능성 있음
- 태을육임 점술(모드2)이 세국 승산지수를 그대로 사용하고 있어, 점시(占時) 기반 개인화된 주객산 산출이 미구현
- 비즈니스 리포트(모드4)가 세국 기반 공통 결과만 출력하며, 개인화 승산지수가 아직 반영되지 않음
- 건강 분석에서 월간(月干)·일간(日干)·시간(時干)의 오행이 오장 판정에 반영되지 않음 (현재 년간만 사용)
- 10년 예측에서 대운(大運) 개념이 미반영 — 현재는 매년 세국을 새로 포국하는 방식이며, 사주 기반 대운 흐름과의 교차는 미구현
- 64괘 유년괘 배당이 약식 산출(생년 기반 단순 순환)이며, 원전의 정밀 산출법(점시·월건·호체 등)이 아직 적용되지 않음
